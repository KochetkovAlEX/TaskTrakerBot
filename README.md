# TaskTrakerBot 🚀

Асинхронный Telegram-бот для управления задачами и личной продуктивности. Проект разработан с фокусом на высокую производительность и корректную работу с конкурентными запросами.

## 🛠 Технологический стек
* **Language:** Python 3.10+
* **Framework:** Aiogram 3.x (Asyncio)
* **Database:** SQLite / MySQL (поддержка через SQLAlchemy)
* **Architecture:** Pattern-based development (ООП).

## 🌟 Ключевые особенности
* **Полная асинхронность:** Эффективная обработка пользовательских действий без блокировки основного потока.
* **State Management:** Использование FSM (Finite State Machine) для реализации сложных многошаговых диалогов.
* **Управление задачами:** Система создания, отслеживания и приоритизации личных дел в реальном времени.
* **Безопасность:** Надежная работа с секретами через переменные окружения (`.env`).

## 🏗 Архитектура проекта
Проект организован по принципу разделения ответственности (Separation of Concerns):
* `handlers/` — логика обработки входящих команд и сообщений.
* `database/` — асинхронный слой взаимодействия с данными.
* `keyboards/` — динамическая генерация интерфейса (Reply/Inline).


## 🚀 Быстрый старт

**Клонируйте репозиторий:**
```bash
git clone https://github.com/KochetkovAlEX/TaskTrakerBot.git
```
**Настройте окружение:**
Создайте файл .env в корне проекта и добавьте ваш токен: 
```TOKEN=your_telegram_bot_token_here```

**Установите зависимости:**
```bash
pip install -r requirements.txt
```
Запустите бота:

```bash 
python bot.py
```
